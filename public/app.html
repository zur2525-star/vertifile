Help & Support

Frequently Asked Questions

What happens if someone alters a protected document?
If a document has been altered after Vertifile protection was applied, the verification will fail immediately. The holographic stamp freezes red and displays a "FORGED" warning.
Was this helpful? Thanks!
What is a PVF file?
A PVF (Protected Verification File) is Vertifile's proprietary format that bundles your original document with cryptographic verification data and a holographic stamp.
Was this helpful? Thanks!
How does Vertifile protect my documents?
Vertifile creates a unique cryptographic hash and embeds it with a holographic verification stamp containing blockchain-backed digital signatures.
Was this helpful? Thanks!
What file types are supported?
Vertifile supports PDF, images (PNG, JPG, TIFF), text files (TXT), and Word documents (DOCX). Maximum file size is 50MB.
Was this helpful? Thanks!
How do I share a protected document?
From the Documents tab, use the actions menu on any document and select "Copy Share Link." Track opens from your dashboard.
