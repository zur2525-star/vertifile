Alternative

Open PVF Files Directly

Set your system to always open .pvf files in your browser

macOS
Safari, Chrome, or any browser
  1. Right-click on any .pvf file in Finder
  2. Select Open With and choose Safari or Chrome
  3. Check "Always Open With" to set as default
Windows
Chrome, Edge, or Firefox
  1. Right-click on any .pvf file in File Explorer
  2. Click Open with then Choose another app
  3. Select Chrome or Edge from the list
  4. Check "Always use this app" and click OK
