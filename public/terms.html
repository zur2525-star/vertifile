1. Service Description

Vertifile provides a document verification and protection platform that creates tamper-proof .pvf files using cryptographic signatures and blockchain timestamping. Our service includes:

  • Document protection via SHA-256 hashing and HMAC signatures
  • Blockchain timestamp registration on the Polygon network
  • Live animated verification stamps embedded in .pvf files
  • Document verification through our web interface and API
  • A browser-based .pvf file viewer

Vertifile uses BLIND processing: we never read, access, or store the content of your documents. Only cryptographic hashes are processed.
