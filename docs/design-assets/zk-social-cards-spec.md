```

|                                                          |
|    הצפנה מלאה מקצה לקצה.                                  |
|    השרת לא יכול לקרוא.                                    |
|    (Heebo, Regular, 24px, #C4B5FD)                       |

|    [אלמנט ויזואלי: מגן/מנעול]                             |
|    מגן סגול (#7C3AED) עם gradient פנימי                   |
|    בתוכו: סמל מנעול סגור בלבן                             |
|    מסביב: קשתות מרוחקות (orbits) בסגול בהיר              |
|    שמסמלות הגנה רב-שכבתית                                 |
|    גודל: ~300x300px, מרכוז אנכי                          |
|                                                          |
|  [תחתית - רצועה]                                          |
|    [שמאל: לוגו Vertifile לבן, 40px גובה]                 |
|    [ימין: vertifile.com, Inter, 20px, #9CA3AF]           |

```

### הערות עיצוב

- כיוון טקסט RTL, הטקסט העברי מיושר לימין
- Badge הטכנולוגיות בשורה אחת, מופרד בקו אנכי
- המגן צריך להיות נקי, מודרני, לא ריאליסטי -- סגנון line art או flat
- gradient עדין ברקע: מ-#1E1B2E (מימין למעלה) ל-#2D1B4E (משמאל למטה)
- ללא אימוג'י. כל האייקונים כ-SVG בלבד

- ללא אימוג'י, ללא צילומים, רק איורים vector

## הנחיות כלליות

### Safe Zone
- שוליים של 60px מכל צד -- שום טקסט או אלמנט חשוב לא חותך את הקצוות
- אלמנטים עיקריים בשליש המרכזי (vertical) -- פלטפורמות חותכות לפעמים את הקצוות

### קבצים לייצוא
- PNG, 72dpi, sRGB
- בנוסף: גרסת 2x (2400px רוחב) לתצוגה retina
- שמות קבצים:
  - `zk-card-linkedin-1200x627.png`
  - `zk-card-twitter-1200x675.png`
  - `zk-card-og-1200x630.png`

### נגישות
- יחס ניגוד טקסט לרקע: מינימום 4.5:1 (WCAG AA)
- סגול #7C3AED על #1E1B2E = 4.6:1 (עובר)
- לבן #FFFFFF על #1E1B2E = 15.4:1 (עובר)
- #C4B5FD על #1E1B2E = 7.2:1 (עובר)

### alt text מוצע
- LinkedIn: "Vertifile Zero-Knowledge Encryption -- המסמכים שלכם רק שלכם. הצפנה מקצה לקצה עם AES-256-GCM."
- Twitter: "Vertifile -- המסמכים שלכם רק שלכם. Zero-Knowledge Encryption."
- OG: "Vertifile -- אפס ידע, מאה אחוז שליטה. הצפנת Zero-Knowledge לאימות מסמכים."
