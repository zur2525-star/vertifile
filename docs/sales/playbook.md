# Vertifile — Sales Playbook

## תסריט דמו (20 דקות)

### דקות 1-3: הבעיה
"כמה פעמים קיבלתם מסמך והייתם צריכים לוודא שהוא אמיתי? כמה זמן זה לקח?"
→ הציגו סטטיסטיקות: 27% ממנהלי HR נתקלים בתעודות מזויפות

### דקות 3-8: הדמו
1. פתח vertifile.com
2. העלה PDF דמו (דיפלומה)
3. הראה את תהליך היצירה (10 שניות)
4. פתח את ה-.pvf — הראה את החותמת מסתובבת
5. "זו חותמת חיה. היא מסתובבת כל הזמן. אם מישהו ישנה אפילו אות אחת..."

### דקות 8-12: הזיוף
1. פתח את אותו מסמך — שנה ציון/שם
2. הראה: חותמת קפואה, X אדום, "FORGED"
3. "ככה כל אחד יכול לדעת תוך שנייה אם המסמך אמיתי"

### דקות 12-16: הטכנולוגיה
- "אנחנו לא קוראים את המסמך שלכם — BLIND processing"
- "9 שכבות אבטחה"
- "פטנט רשום, IANA MIME type"
- "עובד בכל דפדפן, בלי התקנה"

### דקות 16-20: CTA
- "רוצים לנסות? פיילוט חינם ל-3 חודשים"
- "100 מסמכים, ללא עלות, ללא התחייבות"

## 10 התנגדויות + תגובות

| # | התנגדות | תגובה |

| 2 | "זה מוכר משפטית?" | "הגשנו פטנט ורישום IANA. ה-.pvf מספק ראיה קריפטוגרפית חזקה. בעתיד קרוב נוציא חוות דעת משפטית." |
| 3 | "מה אם אתם נסגרים?" | "קבצי .pvf הם HTML עצמאיים — ימשיכו לעבוד. רשומות blockchain נשארות לנצח. אנחנו מחויבים ל-90 יום הודעה מוקדמת." |
| 4 | "יקר מדי" | "Pro ב-$49 לחודש ל-500 מסמכים = פחות מ-10 אגורות למסמך. כמה עולה לכם שיחת טלפון לאימות?" |
| 5 | "אין לנו תקציב" | "פיילוט חינם ל-3 חודשים. תראו את הערך לפני שמשלמים." |
| 6 | "אנחנו משתמשים ב-QR codes" | "QR code קל לזייף — מדפיסים חדש מעל הישן. חותמת הולוגרפית חיה לא ניתנת לזיוף." |
| 7 | "לא מכירים את הפורמט" | "זה כמו PDF — נפתח בכל דפדפן. הגשנו רישום IANA כמו שPDF הוגש בזמנו." |
| 8 | "מה לגבי GDPR?" | "אנחנו לא קוראים את המסמך בכלל. BLIND processing = privacy by design. יש לנו DPA למי שצריך." |
| 9 | "צריך אישור IT" | "מושלם. שלחו את ה-CTO שלכם לvertifile.com/integration — תיעוד API מלא, SDK ל-Node.js ו-Python." |
| 10 | "נחזור אליכם" | "בשמחה. אני שולח לכם one-pager עם כל הפרטים + קישור לדמו חי. מתי נוח לשיחת follow-up?" |

## רצף מיילי Follow-Up

### יום 1 (אחרי הדמו)
נושא: "תודה על הדמו — הנה הגישה לפיילוט"
תוכן: תודה, קישור ל-signup, one-pager מצורף

### יום 3
נושא: "שאלה קצרה"
תוכן: "הספקתם לנסות? אשמח לעזור עם האינטגרציה"

### יום 7
נושא: "Case study: איך [מוסד דומה] משתמש ב-Vertifile"
תוכן: שלח case study רלוונטי

### יום 14
נושא: "הפיילוט שלכם מסתיים בעוד חודשיים"
תוכן: "רוצים להאריך או לשדרג ל-Pro?"
