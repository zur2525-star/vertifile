# Vertifile — חוקי פיד וסושיאל

## עיקרון עליון
הפיד = הפנים של החברה. כל פוסט חייב להיראות כאילו חברה מקצועית עם צוות גרפי עשתה אותו. אפס פוסטים גנריים. אפס "שגר ושכח".

## איך חושבים על פוסט (חובה!)
1. **תחקור קודם** — לפני שכותבים מילה, חפש נתונים אמיתיים, מקורות, מחקרים
2. **הנתון הוא הפוסט** — מצאת סטטיסטיקה מעניינת? זה הפוסט. לא להמציא מספרים!
3. **טקסט + תמונה = דבר אחד** — לא חושבים על טקסט בנפרד ותמונה בנפרד. הרעיון נולד שלם
4. **הסוכנים חושבים ביחד** — נועה, שירה, ליאור, דנה, גל — כולם חושבים על התוצאה הסופית, לא רק על התחום שלהם
5. **כל נתון חייב מקור** — שם המחקר, שנה, מי פרסם. אין "לפי מחקרים" סתמי
6. **פשוט > מסובך** — מספר גדול על רקע נקי > infographic עמוס

```

```

גלים סגולים עדינים ברקע

### 3. QUOTE/MESSAGE — מסר טקסטואלי
```
רקע: לבן עם gradient סגול עדין בצד
מרכז: ציטוט/מסר בגדול (Inter Bold)

```

### 4. EDUCATIONAL — הסבר/infographic
```

כותרת ברורה למעלה
```

### 5. CTA POST — קריאה לפעולה
```
רקע: gradient סגול (#4f46e5 → #7c3aed)
טקסט לבן bold
כפתור/badge "Try Free"

```

## תהליך יצירת פוסט (חובה!)

### שלב 1 — רעיון (נועה + שירה → ליאור)
1. נועה מחליטה: נושא, keyword, angle, קהל יעד
2. שירה כותבת: hook, body, CTA, hashtags — **לכל פלטפורמה בנפרד**
3. שירה מסבירה לליאור: "הרעיון הוא X. המסר הוא Y. הטון הוא Z."
4. ליאור מבינה את הרעיון לעומק לפני שממשיכים

### שלב 2 — עיצוב (ליאור → דנה + גל)
1. ליאור מסבירה לדנה וגל: "הרעיון של הפוסט הוא X. המסר הוא Y."
2. דנה מחליטה: layout, visual hierarchy, מה בולט
3. גל בודק: צבעים תואמים? פאטרן נכון? font נכון? מידות?
4. ביחד כותבים prompt מפורט ל-NanoBanana
5. **גל מאשר** שה-prompt עומד בחוקים הגרפיים

### שלב 3 — יצירה (ליאור)
1. שולחת prompt ל-NanoBanana
2. **בודקת את התמונה** — עומדת בחוקים?
   - צבעי מותג? ✓
   - רקע נכון? ✓
   - לא עמוס? ✓
   - טקסט קריא? ✓
   - מקצועי? ✓
3. אם לא טוב → מתקנת prompt → מייצרת שוב (עד 3 ניסיונות)
4. אם טוב → ממשיכה

### שלב 4 — פרסום (ליאור)
1. מכינה גרסה **שונה** לכל פלטפורמה:
   - LinkedIn: professional, 200-300 words, 1200x627
   - Twitter: punchy, 280 chars, 1600x900
   - Facebook: conversational, 100-200 words, 1200x630
   - Instagram: visual-first, 1080x1350, 25 hashtags
2. **לא אותו טקסט בכל מקום!**
3. מפרסמת דרך Zernio

### שלב 5 — שמירה
שומרת ב-docs/social-media/daily/YYYY-MM-DD.md:
- הרעיון (מנועה+שירה)
- ה-brief הויזואלי (מדנה+גל)
- ה-prompt ל-NanoBanana
- הטקסט לכל פלטפורמה
- לינק לתמונה
