# Competitive Snapshot
**Author:** Amit (Product Manager)
**Date:** 2026-04-11
**Status:** Living document -- updated quarterly

---

## 1. Market Positioning

Vertifile operates in the **document anti-forgery / document integrity** space. We are NOT a digital signature platform (DocuSign, Adobe Sign) or a document management system (SharePoint, Box). We complement both.

**Our unique angle:** The only platform that creates self-contained tamper-proof files (.pvf) with live animated verification stamps, zero-knowledge processing, and blockchain anchoring -- all without ever reading document content.

---

## 2. Direct Competitors

| Competitor | What They Do | Our Advantage |
|---|---|---|
| **DocuSign** | E-signatures, contract lifecycle management | DocuSign proves WHO signed. Vertifile proves the document ITSELF is unaltered. Complementary, not competitive. |
| **Adobe Sign** | E-signatures within Acrobat ecosystem | Same as DocuSign. PDF signatures can be stripped/forged. PVF stamps cannot. |
| **Notarize.com** | Remote online notarization | Notarization proves a person appeared. Vertifile proves document integrity post-notarization. |
| **Certinal** | Cloud-based e-signatures (India market) | Regional player. No anti-forgery, no blockchain, no animated stamps. |
| **Secured Signing** | Digital signatures with compliance | Signature-only. No tamper detection, no file format protection. |

## 3. Adjacent Competitors (Partial Overlap)

| Competitor | Overlap Area | Differentiation |
|---|---|---|
| **Hyperledger / Blockchain cert platforms** | Blockchain verification | Generic blockchain tools require dev effort. Vertifile is turnkey. |
| **Accredible / Credly** | Digital credentials for education | Credential-specific. Vertifile handles ANY document type. |
| **TrueScreen** | Screenshot/content authentication | Screenshot-only. No document protection, no .pvf format. |

---

## 4. Key Differentiators We Own

1. **BLIND Processing** -- server never reads content. No competitor offers this.
2. **Live Animated Stamp** -- impossible to screenshot-forge. Unique to Vertifile.
3. **.pvf File Format** -- self-contained, opens in any browser. Patent pending.
4. **Instant Forgery Detection** -- stamp freezes red in real-time if tampered.
5. **No Software Required** -- .pvf files open in any browser. Zero installation.

---

## 5. Competitive Threats

- **Adobe** could add tamper-detection to PDF. Risk: medium. Timeline: 12-18 months minimum.
- **DocuSign** could add document integrity features. Risk: low. They are focused on CLM.
- **Blockchain-native startups** could emerge. Risk: medium. Our head start is the UX + file format.

---

## 6. Positioning Statement

**For organizations that issue, receive, or archive critical documents**, Vertifile is the **document anti-forgery platform** that proves any document is authentic and unaltered -- without ever reading its content. Unlike e-signature tools that only prove who signed, Vertifile proves the document itself has not been tampered with, using cryptographic verification, blockchain anchoring, and a live animated stamp that is impossible to forge.
