## Part 3 -- 9 Security Layers Explained (1:30 - 2:30)

### Visual
Animated vertical stack of nine layers building on top of each other. Each layer lights up as it is explained. Use icons and simple labels. Do not overwhelm -- keep it visual.

### Voiceover
"What makes Vertifile different from a simple digital signature? Depth. Every PVF file is protected by nine independent security layers."

**Layer explanations (cycle through with ~6 seconds each):**

1. **Cryptographic Hash** -- "A unique digital fingerprint of the entire document. Change one byte, the fingerprint breaks."

2. **Timestamp Certification** -- "A tamper-proof record of exactly when the document was created."

3. **Origin Verification** -- "Confirms who created the document and where it came from."

4. **Content Integrity Check** -- "Verifies every page, every image, every character matches the original."

5. **Metadata Validation** -- "Checks hidden document properties for signs of manipulation."

6. **Format Consistency** -- "Ensures the file structure has not been altered or reconstructed."

7. **Visual Authentication** -- "The animated stamp provides instant, visible proof of status."

8. **Chain of Custody** -- "Tracks the verification path from creation to the current check."

9. **Cross-Reference Validation** -- "Compares against the original record stored securely by Vertifile."

### Closing line
"Each layer works independently. All nine must pass for the document to be verified. That is what makes forgery practically impossible."
